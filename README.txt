Squish-Mate - Ready to Run

Squish-Mate is a lightweight, cross-platform desktop pet that monitors your activities and interacts with speech bubbles.

To run the application:

1. Setup virtual environment:
   python3 -m venv --system-site-packages .venv
   .venv/bin/pip install PySide6 psutil requests pynput Pillow

2. Run the pet:
   ./run_pet.sh
   # or: .venv/bin/python desktop_pet.py

For full installation and configuration steps, please refer to:
- README.md (General overview and file structure)
- USAGE.md (Prerequisites, system setup, and Ollama integration details)

The project is located at: ~/Projects/Personal/desktop-pet/
All components are functional and ready to be run.
